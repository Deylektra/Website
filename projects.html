CAPtivate
This was my Senior Project at Western. My team and I created a software for the Civil Air Patrol. During this project we recieved experience in creating documentation as well as scrum. The software helps with tracking attendance and improve safety at Civil Air Patrol meetings.
How I contributed: I created the User Interface for members to use and manage the tasks that was required. I used Java Swing to create the front end. While creating the front end, my teammate and I used Model-View-Controller model to help keep our code modular. I also got to work on the backend. The client wanted the software to print reports of the attendance, so I worked on the printing functionality.
